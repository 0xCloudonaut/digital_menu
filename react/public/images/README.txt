Place your dish photos in this folder.

How to use your own images:
1) Add image files in /public/images (jpg, jpeg, png, webp).
2) In src/data/menuData.js, set each item's "image" field.
   Example: image: '/images/my-dish-photo.jpg'
3) Keep the leading slash (/images/...) so Vite serves it from /public.

Current expected file names in menuData.js:
- margherita-pizza.jpg
- paneer-tikka-wrap.jpg
- grilled-salmon-bowl.jpg
- chicken-dumplings.jpg
- chocolate-lava-cake.jpg
- fresh-lime-soda.jpg

If any file is missing, the app automatically shows a fallback placeholder image.
